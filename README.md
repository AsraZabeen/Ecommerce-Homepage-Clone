# Ecommerce-Homepage-Clone

## overview
A frontend clone of the Flipkart homepage built using HTML and CSS. This project was created to practice web page structuring, styling, layout design, and responsive user interface development.

## Features
>> Homepage layout inspired by flipkart
>> Navigation bar
>> Product categories section
>> Promotional banners
>> Product display cards
>> Clean and organized user interface
>> Responsive design elements

## Technologies Used
* HTML
* CSS

## Project purpose
    This project was developed to improve frontend web development skills and gain practical experience in designing real
world website layouts using HTML and CSS    

## Files Included
* index.html
* style.css
* responsive.css
* utils.css
* logo.png
